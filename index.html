Movie Night Picker — AI Agent

Designed, deployed, and later rebuilt a live AI recommendation agent on Agent.ai integrating GPT-5 mini. After auditing the original version, redesigned the intake flow and output structure end-to-end, then implemented the upgrade directly in the no-code builder — solving real decision fatigue around "what should we watch tonight."

  • Structured, one-question-at-a-time intake — mood, who's watching, time available, genres, and streaming platforms — instead of one open-ended prompt
  • Returns three ranked picks per session, each with a short pitch, why it fits tonight, runtime, and a content note
  • Side-by-side comparison table so users can weigh all three picks at a glance
  • Deliberate design trade-off: skipped direct streaming deep-links (LLMs can't reliably track live catalog data) in favor of JustWatch search links — a call grounded in understanding AI's limitations
  • Built-in feedback loop — if none of the picks land, the agent invites another round on the spot
